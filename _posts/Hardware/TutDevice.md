# 示波器

## 使用步骤
#### 第1步:基本准备与连接
1. 开机:连接电源,按下电源键
2. 探头设置(非常关键！):
  1. 调节衰减开关,1X,10X ,一般选择10X
     1. 减少对被测电路的负载影响(负载效应)
     2. 扩大示波器的电压测量范围
     3. 提供更高的带宽
3. 探头补偿(新探头或长时间未使用时必须做):
  1. 将探头尖端钩到示波器前面板提供的 "补偿信号输出端"(通常是一个方波输出,约1kHz)
  2. 将探头地线(夹子)接到补偿信号的地端
  3. 在屏幕上观察波形一个正确补偿的探头应显示完美的方波
  4. 过补偿:方波波形边沿出现圆角
  5. 欠补偿:方波波形边沿出现尖峰
  6. 如果波形不完美,需要使用调节棒调节探头上的补偿电容,直到波形完美这一步没做,所有测量都可能不准确


#### 第2步:连接被测电路
1. 接地点:必须先接地！ 将探头的地线夹子夹到电路板的地(GND) 上地是测量的电压参考点,不接地或接地不良会导致波形畸变、乱跳、有噪声
2. 测信号点:将探头的尖端接触到您要测量的点,例如SPI的CLK引脚
3. 多通道测量:如果要同时看CLK和MOSI(就像您在排查中做的那样),请将第二个探头连接到CH2,地线夹子夹到同一个地,尖端接触MOSI引脚

#### 第3步:调整示波器至稳定显示波形
1.  自动设置(快捷键 AutoScale / AutoSet):
2.  手动调整(理解原理):
    - 垂直系统(控制Y轴,电压):
        - 通道开关:确保CH1、CH2是开启状态(屏幕亮起)
        - 电压刻度(Volts/Div):旋转此旋钮,调整屏幕上每一大格代表的电压值(如 2V/格,1V/格)目标是让波形的幅度占据屏幕的1/2到2/3,既不要太小看不清,也不要超出屏幕
    - 水平系统(控制X轴,时间):
        - 时间刻度(Time/Div,秒/格):旋转此旋钮,调整屏幕上每一大格代表的时间(如 1ms/格, 500μs/格)目标是能在屏幕上稳定显示1到3个完整的波形周期
    - 触发系统(让波形"静止"的关键):
        - 触发源(Source):选择你要观测的哪个通道作为触发源例如,要观测SPI,通常选择CLK(接在CH1) 作为触发源,因为它是最规律的周期性信号
        - 触发边沿(Slope):选择上升沿(↗)或下降沿(↘)通常选上升沿
        - 触发电平(Level):旋转此旋钮,屏幕上会看到一条横线(触发电平线)将其调整到被测信号电压范围的中间值例如,一个0-3.3V的信号,将触发电平设在1.6V左右示波器会持续采集数据,但只有当信号电压达到你设定的这个电平和边沿条件时,它才会开始绘制一帧波形 这就是波形稳定的秘诀



# 逻辑分析仪和示波器区别

示波器用于分析信号的模拟特性,而逻辑分析仪用于分析数字系统的逻辑状态和时序关系


## 具体区别

1. 示波器 是一种电压-时间测量仪器它的首要任务是精确复现和测量信号在时域上的连续电压变化它关注的是信号的模拟保真度,包括电压幅度、上升/下降时间、过冲、振铃、噪声等参数
2. 逻辑分析仪 是一种逻辑状态-时间测量仪器它的设计目标是捕获和显示多路数字信号的逻辑状态(高或低)及其时序关系它本质上是一个多通道的数字信号采样器,只关心信号是否超过设定的逻辑门限电压,而不关心其具体的模拟形状



| 特性     | 示波器                                            | 逻辑分析仪                                                  |
| :------- | :------------------------------------------------ | :---------------------------------------------------------- |
| 核心功能 | 观察信号的模拟特性(电压、波形、噪声)              | 观察信号的数字逻辑(逻辑电平、时序关系、协议)                |
| 显示内容 | 连续的电压-时间波形                               | 并行的时间-逻辑状态表格(波形图)                             |
| 关键指标 | 带宽、采样率、垂直分辨率                          | 通道数量、定时采样率、协议支持                              |
| 通道数量 | 通常较少(2/4通道)                                 | 非常多(8, 16, 32, 甚至上百通道)                             |
| 输入电路 | 高阻抗,可精确测量电压幅度                         | 阈值比较器,只判断高/低电平                                  |
| 触发方式 | 边沿、脉宽、斜率、视频等                          | 边沿、模式(特定逻辑组合)、协议触发(如I2C的起始位)           |
| 主要用途 | 调试模拟电路、电源质量、信号完整性、EMC预兼容测试 | 调试数字电路、分析通信总线、分析多路信号时序、FPGA/CPLD验证 |
| 价格     | 同等性能下通常更贵                                | 相对便宜,特别是USB接口的逻辑分析仪                          |



示波器 显示为连续的电压波形调试应用侧重于:
1. 信号完整性分析(SI)
2. 电源完整性分析(PI),如测量电源噪声和纹波
3. 模拟电路特性分析
4. 验证和调试传感器等模拟前端

逻辑分析仪 显示为并行的逻辑电平波形(定时图)或协议列表(如十六进制、二进制)调试应用侧重于:
1. 数字通信协议解码与分析(如I2C, SPI, UART, CAN等)
2. 多路数字信号之间的时序关系验证
3. 硬件与软件协同调试,验证代码执行产生的硬件逻辑是否正确
4. FPGA/CPLD的逻辑验证
