> 本文以惠普笔记本为例，BIOS 启动键为 `F10`，其他品牌请自行查阅（如联想为 `F2`）。

## 一、准备工作

- **系统镜像**：Ubuntu 20.04.1 LTS（桌面版，`ubuntu-20.04.1-desktop-amd64.iso`）
- **刻录软件**：UltraISO（官网：[https://cn.ultraiso.net](https://cn.ultraiso.net)）
- **U 盘**：容量 ≥ 4GB（制作过程会格式化，请提前备份数据）
- **硬盘空间**：建议预留 **60GB 以上** 的未分配空间

> 镜像可从[清华大学开源镜像站](https://mirrors.tuna.tsinghua.edu.cn/)下载。

## 二、检查电脑状态

### 1. 查看 BIOS 模式（UEFI / Legacy）

- 按 `Win + R` → 输入 `msinfo32` → 回车
- 查看 **BIOS 模式**：若为 `UEFI`，则后续安装需对应 UEFI 模式启动

### 2. 查看磁盘类型

- 右键“此电脑” → 管理 → 磁盘管理
- 确认磁盘类型为 **基本磁盘**（动态磁盘可能无法顺利安装双系统）

## 三、制作 Ubuntu 启动 U 盘

1. 插入 U 盘（无需提前手动格式化，写入镜像时会自动处理）
2. 打开 UltraISO → “文件” → “打开” → 选择下载的 Ubuntu 镜像
3. 点击菜单栏 **启动** → **写入硬盘映像**
4. 设置：
   - **硬盘驱动器**：选择你的 U 盘（注意不要选错）
   - **写入方式**：推荐 `USB-HDD+`
5. 点击 **写入** → 等待完成

> **关于 U 盘占用空间**：制作完成后，U 盘已用空间 ≈ 镜像文件大小。例如，`ubuntu-20.04.1-desktop-amd64.iso` 大小为 2.59 GB，写入后 U 盘占用即为 2.59 GB。

> **手动格式化 U 盘的方法**（可选）：  
> 打开“我的电脑” → 右键点击 U 盘 → 选择“格式化” → 设置文件系统为 `FAT32`，分配单元大小为 `4096 字节`，勾选“快速格式化” → 点击“开始”。

## 四、为 Ubuntu 分配磁盘空间

1. 打开 **磁盘管理**
2. 选择一个空闲空间较大的分区（或直接使用已删除卷的空闲区域）
3. 若磁盘已满，可右键分区 → **压缩卷**，压缩出 60GB 以上空间
4. 若已有专门分区，可直接 **删除卷**，得到 **未分配区域**

> 无需在 Windows 下对新分区进行格式化，保持为“未分配”状态即可。

## 五、修改 BIOS 启动顺序

1. 重启电脑，开机时按对应键进入 BIOS：
   - 惠普：`F10`  
   - 联想：`F2`  
   - 戴尔：`F2` 或 `F12`  
   - 华硕：`F2` 或 `Del`
2. 找到 **Boot** 或 **启动** 选项卡
3. 将 U 盘（通常显示为 USB HDD 或 U盘名称）设为第一启动项
4. 保存并退出（通常按 `F10` 保存）

## 六、安装 Ubuntu 20.04

1. 从 U 盘启动后，选择 **Try Ubuntu** 或直接 **Install Ubuntu**
2. 选择语言 → 键盘布局 → 选择 **正常安装**（可勾选“安装第三方软件”）
3. **安装类型**：选择 **其他选项**（手动分区）
4. 找到之前预留的 **未分配空间**，按以下建议分区（UEFI 模式）：

| 挂载点 | 大小 | 类型 | 说明 |
|--------|------|------|------|
| （无） | 512 MB | EFI 系统分区 | 已有 EFI 分区可共用，无需新建 |
| / | 20~30 GB | ext4 日志文件系统 | 系统根目录 |
| swap | 与内存大小相当（如 8GB） | 交换空间 | 用于虚拟内存 |
| /home | 剩余全部 | ext4 日志文件系统 | 用户数据目录 |

> 若为 Legacy 模式，则不需要 EFI 分区，需创建一个 `/boot` 分区（约 1GB）并安装引导到主硬盘。

5. 选择 **安装启动引导器的设备**：默认为主硬盘（如 `/dev/sda`）即可
6. 设置用户名、密码等，等待安装完成

## 七、附录：选项详解

### 1. UltraISO 写入方式说明

| 方式 | 说明 |
|------|------|
| USB-HDD | 模拟硬盘模式，支持分区和启动 |
| USB-ZIP | 模拟 ZIP 驱动器，兼容性较差 |
| USB-HDD+ | 增强版，支持更大容量和更快速度 |
| USB-ZIP+ | 增强版 ZIP 模式 |
| RAW | 原始写入，不支持分区和启动 |

> 制作 Ubuntu 启动盘推荐 `USB-HDD+`。

### 2. U 盘格式化选项

- **FAT32**：兼容性最好，但单个文件不能超过 4GB
- **NTFS**：支持大文件，但在部分 Linux/Mac 上只读
- **exFAT**：适合跨平台大文件传输，但老旧系统可能需驱动

> 制作启动盘时，UltraISO 会自动格式化，无需手动干预。若需提前手动格式化，可按上文“手动格式化 U 盘的方法”操作。

### 3. 分配单元大小（簇大小）

- 默认 **4096 字节** 即可，适合绝大多数 U 盘使用
- 更大的簇（如 64KB）可提升大文件读写性能，但会浪费少量空间

## 八、常见问题与提示

- **关闭 Windows 快速启动**：进入控制面板 → 电源选项 → 选择电源按钮功能 → 取消“启用快速启动”，避免引导冲突。
- **关闭安全启动（Secure Boot）**：部分电脑需在 BIOS 中关闭，否则 Ubuntu 可能无法正常引导。
- 若安装后无 Windows 启动项，可在 Ubuntu 终端执行 `sudo update-grub` 更新引导菜单。
- 建议安装完成后，将 BIOS 中的启动顺序改回硬盘优先。
