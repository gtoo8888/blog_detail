# 正则表达式教程

## 一、基础语法

| 字符 | 描述 |
|:---:|---|
| `\` | 转义下一个字符 |
| `^` | 匹配字符串开始位置 |
| `$` | 匹配字符串结束位置 |
| `*` | 匹配前面的子表达式零次或多次 |
| `+` | 匹配前面的子表达式一次或多次 |
| `?` | 匹配前面的子表达式零次或一次 |
| `.` | 匹配除换行符 `\n` 外的任意单个字符 |
| `x\|y` | 匹配 x 或 y |
| `[xyz]` | 字符集合，匹配其中任意一个字符 |
| `[^xyz]` | 负值字符集合，匹配未包含的任意字符 |
| `[a-z]` | 字符范围，匹配 a 到 z 之间任意字符 |
| `[^a-z]` | 负值字符范围，匹配不在 a 到 z 之间的字符 |

## 二、转义字符

| 字符 | 描述 | 等价 |
|:---:|---|---| 
| `\d` | 匹配数字字符 | `[0-9]` |
| `\D` | 匹配非数字字符 | `[^0-9]` |
| `\s` | 匹配空白字符（空格、换页、换行、回车、制表、垂直制表） | `[ \f\n\r\t\v]` |
| `\S` | 匹配非空白字符 | `[^ \f\n\r\t\v]` |
| `\w` | 匹配单词字符（字母、数字、下划线） | `[A-Za-z0-9_]` |
| `\W` | 匹配非单词字符 | `[^A-Za-z0-9_]` |
| `\b` | 匹配单词边界 | |
| `\B` | 匹配非单词边界 | |

## 三、控制字符

| 字符 | 描述 | 等价 |
|:---:|---|---|
| `\cx` | 匹配由 x 指明的控制字符 | `\x0c` 和 `\cL` |
| `\f` | 匹配换页符 | `\x0c` |
| `\n` | 匹配换行符 | `\x0a` |
| `\r` | 匹配回车符 | `\x0d` |
| `\t` | 匹配制表符 | `\x09` |
| `\v` | 匹配垂直制表符 | `\x0b` |

## 四、修饰符

| 修饰符 | 描述 |
|:---:|---|
| `i` | 忽略大小写匹配 |
| `g` | 全局匹配，找到所有符合的子串，而非仅第一个 |
| `m` | 多行模式，`^` 和 `$` 匹配每行起始/结束位置 |
| `s` | 单行模式，`.` 可匹配换行符 |
| `u` | 启用 Unicode 匹配 |
| `y` | 粘滞模式，只从上次匹配位置继续 |
