# kdevelop
CMake可视化调试工具


打开命令窗口:
1. 安装gcc(编译器)
    sudo apt-get build-dep gcc
    sudo apt-get install build-essential
    
2. 安装kdevelop
sudo apt-get install kdevelop

3. 安装cmake等
sudo apt-get install automake autoconf g++ libtool cmake
